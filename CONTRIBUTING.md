# Contributing

Contributions welcome! This is an open source project — MIT licensed.

## Setup

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/hospital-bill-checker
cd hospital-bill-checker

# Install dependencies
bun install

# Copy env
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY

# Build CMS lookup data
pip3 install openpyxl
python3 scripts/build_mpfs.py
python3 scripts/build_ncci.py
python3 scripts/build_asp.py
python3 scripts/build_hospital_index.py

# Run dev server
bun run dev
```

## Architecture

```
src/
├── routes/
│   ├── +page.svelte          # Main app (upload → processing → results)
│   ├── api/
│   │   ├── audit/+server.ts  # POST /api/audit — main audit endpoint
│   │   ├── parse/+server.ts  # POST /api/parse — PDF extraction
│   │   ├── stats/+server.ts  # GET /api/stats — live counters
│   │   └── live-users/+server.ts # GET /api/live-users — GA4 realtime fallback
│   └── privacy/+page.svelte  # Privacy policy
├── lib/
│   ├── types.ts              # Shared TypeScript types
│   ├── server/
│   │   ├── claude.ts         # Claude API integration + audit logic
│   │   └── pdf.ts            # PDF parsing (pdf-parse + Vision fallback)
│   └── data/                 # CMS lookup JSON (generated by scripts/)
└── app.css                   # Design tokens + global styles

scripts/                      # Python CMS data pipeline (run quarterly)
examples/                     # Synthetic test bills
```

## Error types detected

| Error | Data source | Detection method |
|-------|------------|-----------------|
| Upcoding | Claude reasoning | ICD-10 vs E&M level mismatch |
| Unbundling | NCCI PTP table | Static lookup |
| Pharmacy markup | CMS ASP table | billedAmount / ASP > 4.5x |
| ICD-10 mismatch | Claude reasoning | Clinical coherence check |
| Duplicate billing | Bill data | Same CPT + date dedup |

## Running tests

```bash
bun run test           # unit tests
bun run test:e2e       # playwright e2e
```

## CMS data refresh

The lookup tables in `src/lib/data/` come from public CMS datasets. Refresh quarterly:
```bash
python3 scripts/build_ncci.py   # quarterly
python3 scripts/build_asp.py    # quarterly
python3 scripts/build_mpfs.py   # annually (January)
python3 scripts/build_hospital_index.py   # monthly
```

## Pull requests

- Keep PRs focused — one feature or fix per PR
- Tests required for new audit logic
- Don't include real patient data — use synthetic examples only
- Run `bun run check` before submitting
